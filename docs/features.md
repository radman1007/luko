# قابلیت‌های LUKO — استخراج‌شده از پروژه‌ی قدیمی (LOKO/Front-Codes)

این سند مرجع بازسازی است: هر قابلیتی که نسخه‌ی جدید باید داشته باشد اینجا لیست شده.
پیاده‌سازی هر بخش که تمام شد، جلوی آن ✅ بزنید.

## کلیات محصول
پلتفرم آموزشی-سرگرمی کودکان دبستانی (فارسی، RTL، mobile-first) با گیمیفیکیشن
(سکه، XP، استریک، مدال) و پنل‌های جداگانه برای والدین، معلم، مدرسه و ادمین.

## نقش‌ها و دسترسی
- **student** — تجربه‌ی اصلی اپ (خانه، کتاب‌ها، کلاب، تلویزیون، سلامت، پادکست، پروفایل)
- **parent** — پنل والدین: مدیریت فرزندان + گزارش پیشرفت + تیکت
- **teacher** — پنل معلم: کلاس‌ها و دانش‌آموزان
- **school_admin** — پنل مدرسه: مدیریت کلاس‌ها و معلم‌ها
- **team_admin** — پنل ادمین: مدیریت مدارس، محتوا (ویدیو/پادکست)، تیکت‌ها
- **مهمان (guest)** — بدون حساب؛ داده‌ی دمو (کتاب‌ها و بازی‌های کلاس اول) و سکه در localStorage
- هر نقش بعد از ورود به مسیر خودش redirect می‌شود؛ مسیرها با ProtectedRoute محافظت می‌شوند

## احراز هویت
- ورود (شماره موبایل + رمز)، ثبت‌نام والدین، تکمیل پروفایل بعد از ثبت‌نام
- JWT با access token (+ refresh token در بک‌اند)، ذخیره در localStorage
- interceptor توکن روی همه‌ی درخواست‌ها؛ خروج و پاک‌سازی توکن

## خانه (دانش‌آموز)
- کارت اطلاعات کاربر: XP، سکه، استریک روزانه
- مودال ثبت حال (mood) هنگام ورود + یادآور دوره‌ای
- دکمه‌های دسترسی سریع به بخش‌ها
- پیش‌نمایش کتاب‌ها، بنرهای لوکو TV (اسلایدر)، کارت پادکست، بخش دستاوردها
- اعلان‌ها (notifications) با وضعیت دیده‌شده
- نویگیشن پایین (۵ تب: کلاب، تلویزیون، خانه، سلامت، پروفایل)

## کتاب‌ها و بازی‌ها (هسته‌ی آموزشی)
- لیست کتاب‌های درسی هر پایه (ریاضی، علوم، نگارش، قرآن و…)
- هر کتاب → لیست بازی‌های HTML5 با ترتیب مرحله‌ای (بازی بعدی تا اتمام قبلی قفل است)
- اجرای بازی داخل اپ (iframe از public/games) + گزارش اتمام بازی به بک‌اند
- جایزه‌ی سکه برای هر بازی (coin_reward)

## لوکو کلاب (گیمیفیکیشن)
- مأموریت‌های روزانه (لیست تسک + تکمیل و دریافت سکه)
- استریک هفتگی (کارت روزها) و خلاصه‌ی سکه‌ها
- مدال‌ها/نشان‌ها (badges)
- فروشگاه: خرج کردن سکه برای جایزه (redeem)
- ویدیوهای جایزه (مودال پخش)

## لوکو تلویزیون
- کتابخانه‌ی ویدیو با دسته‌بندی، جستجو و فیلتر
- اسلایدر بنر، کارت ویدیو، مودال پخش
- ثبت تعامل (بازدید/لایک)، آخرین ویدیوهای هر دسته

## لوکو پادکست
- لیست پادکست‌ها + پادکست روزانه + دسته‌بندی‌ها
- پخش‌کننده‌ی صوتی

## لوکو سلامت
- ثبت حال (mood check-in) با یادآور هر ۴ ساعت + تاریخچه
- تمرین تنفس هدایت‌شده (فازهای دم/نگه‌داشتن/بازدم با انیمیشن)
- باغچه‌ی مجازی (رشد با انجام کارهای خوب)
- صندوقچه‌ی رازها: یادداشت‌های رمزدار (ساخت/بازکردن با رمز/ویرایش/حذف)
- وضعیت و آمار سلامت (wellbeing)

## پروفایل (دانش‌آموز)
- اطلاعات کاربر، سطح/XP، تنظیمات، خروج

## پنل والدین
- تب‌ها: فرزندان، داشبورد، پیشرفت، تیکت‌ها
- افزودن/حذف فرزند، گزارش هر فرزند (پروفایل دانش‌آموز)

## پنل معلم / مدرسه / ادمین
- معلم: کلاس‌های من و دانش‌آموزان
- مدرسه: مدیریت کلاس‌ها، افزودن معلم به کلاس
- ادمین: CRUD مدارس، آپلود/مدیریت ویدیو و پادکست، مدیریت تیکت‌ها (تغییر وضعیت)

## پشتیبانی (تیکت)
- ساخت تیکت، لیست تیکت‌ها، رشته‌ی پاسخ‌ها (reply)، وضعیت تیکت

## زیرساخت فنی (که در نسخه‌ی جدید هم لازم است)
- axios client واحد با interceptor توکن و مدیریت 401 → [frontend/src/lib/api.ts](../frontend/src/lib/api.ts)
- پروکسی `/api/v1` در dev (Vite) و nginx در production؛ Docker برای دیپلوی
- lazy loading همه‌ی صفحه‌ها + PageLoader
- RTL کامل + فونت فارسی اختصاصی (قبلاً Shoor)
- Toast، Skeleton loading، اسلایدر (قبلاً Swiper)، انیمیشن (قبلاً framer-motion)
